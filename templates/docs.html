Bulk Mail

This is a simple REST API made for the task of Bulk Emailing from an Excel Sheet.YOU MUST HAVE these colum headings in you excel sheet-'NAME' , 'MAIL' ,'SUBJECT' ,'MESSAGE'

Access

get /multiemail_static/{name}&{mail}&{apppass}&{id}&{sname}

In Excel Sheet MESSAGE and SUBJECT columns can NOT have dynamic data on as {A0} , {A1}....

Path parameters

name (required)
Path Parameter — Senders name
mail (required)
Path Parameter — Senders Email id
apppass (required)
Path Parameter — Gmail App Key (generated from gmail settings)
id (required)
Path Parameter — Google docs ID (obtained from share to all link)
sname (required)
Path Parameter — Sheet name in Google docs 

get /multiemail_dynamic/{name}&{mail}&{apppass}&{id}&{sname}

Excel Sheet should have Column Headings as A0, A1, A2.... and MESSAGE and SUBJECT columns can have dynamic data as {A0} , {A1}....

Path parameters

name (required)
